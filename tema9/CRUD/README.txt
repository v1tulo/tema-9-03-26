Este é um exemplo de aplicativo CRUD (Create, Read, Update, Delete) básico de tarefas (To-Do List) usando PHP, HTML, CSS, MySQL e PDO. Neste exemplo, você poderá adicionar, visualizar, atualizar e excluir tarefas.

Antes de começarmos, certifique-se de criar um banco de dados MySQL chamado todo_app com uma tabela tasks contendo as colunas id (chave primária autoincremental) e description.
